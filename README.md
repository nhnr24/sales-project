# E-Commerce Sales Dashboard

## 📊 Project Overview
This project analyzes e-commerce sales data using MySQL and Power BI.

## 🛠 Tools Used
- MySQL
- Power BI
- Excel / CSV

## 📂 Dataset
- cleaned_data.csv

## ⚙️ Process
1. Loaded dataset into MySQL
2. Created table and inserted data
3. Performed SQL analysis
4. Built dashboard in Power BI

- Total Profit: 1.48M
- Region-wise sales analysis
- Category-wise profit analysis
